### Функция list
принимает значение итерируемого типа и другие списки, строки и т.д.
### Срезы 
это способ получения части списка при помощи указания индексов готовый срез является новым списком. Для создания среза указывается имя списка и определённые индексы. Всего есть 3 способа правил генерации срезов: 
- Указать конечное значение. То есть от 0 до указанного 
- Указать конечное и начальное значение.
- Указать начальное и конечное значение и шаг.
если требуется использовать один из способов но отсутствует инфа. нужно поставить [].

    Пример:
    peanut.append("1")

- index(item) - индекс конкретного элемента  Сначала указывается имя списка через точку вызывается метод index , в который передаётся item. В результате работы возвращать индекс указанного элемента. Если элементов с одинаковым значит несколько то возвращает индекс первого найденного в списке. Если элемента нет выбрасывает исключение ValueEroro.
- len - длина списка. Сначала вызывается функция лен а ей передаётся конкретный список возвращает длину списка в введи целого числа.
  Пример:
   peanut = [1,2,3]
   print(len(peanut))
 -  min и max Минимальное и максимальное значение в списке. Функции возвещают мин макс знач. из указанного в качестве аргумента списка. 
    Пример:
    peanut = [1.2.3]
    print(max(peanut))
    print(min(peanut))
