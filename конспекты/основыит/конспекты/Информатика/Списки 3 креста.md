### Срезы 
это способ получения части списка при помощи указания индексов готовый срез является новым списком. Для создания среза указывается имя списка и определённые индексы. Всего есть 3 способа правил генерации срезов: 
- Указать конечное значение. То есть от 0 до указанного 
- Указать конечное и начальное значение.
- Указать начальное и конечное значение и шаг.
если требуется использовать один из способов но отсутствует инфа. нужно поставить [].

- index(item) - индекс конкретного элемента  Сначала указывается имя списка через точку вызывается метод index , в который передаётся item. В результате работы возвращать индекс указанного элемента. Если элементов с одинаковым значит несколько то возвращает индекс первого найденного в списке. Если элемента нет выбрасывает исключение ValueEroro.
- len - длина списка. Сначала вызывается функция лен а ей передаётся конкретный список возвращает длину списка в введи целого числа.
  Пример:
   peanut = [1,2,3]
   print(len(peanut))
 -  min и max Минимальное и максимальное значение в списке. Функции возвещают мин макс знач. из указанного в качестве аргумента списка. 
    Пример:
    peanut = [1.2.3]
    print(max(peanut))
    print(min(peanut))
- sorted - Функция вернет новый отсортированный  из итерируемых элементов. Функция имеет два необязательных аргумента, которые должны быть указаны в качестве аргументов ключевых слов.
 - insert- **Insert**, или **Inst**, — клавиша на предназначенная для переключения из режима «вставки» (в этом режиме текстовый редактор вводит текст, начиная с точки вставки, и помещает его правее по странице) в режим «замены» (в этом режиме текст, который вводится с клавиатуры, будет заменять собой уже напечатанный до того текст).
 - extend - Метод `extend()` в Python является одним из встроенных методов для работы со списками. Он позволяет добавлять элементы из одного списка в другой, расширяя исходный список. Этот метод особенно полезен, когда нужно объединить два списка или добавить несколько элементов сразу. В отличие от метода `append()`, который добавляет один элемент в конец списка, `extend()` добавляет все элементы из итерируемого объекта (например, другого списка) по одному.
 - remove - удаление
 - pop - Метод позволяет Необязательный аргумент - индекс `i` по умолчанию равен `-1`. Так что по умолчанию эта операция производит действие с последним элементом последовательности.
 - count - В Python для подсчета количества вхождений элемента в последовательности можно использовать метод _count()_ для списков или строк, а также класс _Counter_ из модуля _collections_.
 - sort - Если функция сравнения `compareFunction` не предоставляется, элементы сортируются путём преобразования их в строки и сравнения строк в порядке следования кодовых точек Unicode. Например, слово "Вишня" идёт перед словом "бананы". При числовой сортировке, 9 идёт перед 80, но поскольку числа преобразуются в строки, то "80" идёт перед "9" в соответствии с порядком в Unicode.
 - reverse - В Python существует несколько способов перевернуть последовательность, такие как использование метода _reverse()_, срезов или функции _reversed()_.
 - copy - Модуль copy в Python предоставляет функциональность для поверхностного и глубокого копирования объектов. Это особенно полезно при работе с изменяемыми объектами, такими как списки или словари, когда изменения в одной копии могут непреднамеренно повлиять на оригинал.
