**Выделяют 3 основных метода сортировки:**

1. Пузырьковая(Самая простая, но и медленная).
Ты смотришь на соседние ряды, алгоритмы и сравниваешь их, если левый больше правого - меняем их местами. Самые тяжелые элементы постоянно всплывают в конце, как пузырьки в газировке.

```Python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

2. Вставками (Как на ладони).

Ты берёшь по одной новой карте из колоды и вставляешь её в нужное место среди уже отсортированных карт в другой руке.

```Python
def insertion_sort(arr):
    n = len(arr)
    for i in range(1, n):
        current_value = arr[i]
        j = i - 1
        while j >= 0:
            if current_value < arr[j]:
                arr[j+1] = arr[j]
                arr[j] = current_value
            j = j - 1
        else:
            break
    return arr
``` [1](https://github.com/ViktorSalimonov/sorting-algorithms)

3. Быстрая(Молния, готов?).

Выбираешь один элемент. Всё, что меньше, кидаешься в левую кучу, что больше - в правую. Потом делаешь то же самое с каждой кучей, пока всё не разберёшь.

```Python
def quick_sort(arr):  
    if len(arr) <= 1:  
        return arr  
    else:  
        pivot = arr[len(arr) // 2]  # Выбираем опорный элемент (можно выбрать любой способ выбора)  
        left = [x for x in arr if x < pivot]  
        middle = [x for x in arr if x == pivot]  
        right = [x for x in arr if x > pivot]  
          
        return quick_sort(left) + middle + quick_sort(right)  
  
# Пример использования:  
arr = [3, 6, 8, 1, 2, 7, 4, 5]  
sorted_arr = quick_sort(arr)  
print(sorted_arr)  

```

*Обычно работает быстрее всех из выше перечисленных, поэтому её чаще всего и используют в компьютерах.*

Также нельзя забывать и про другие 2:

1. Слиянием(Делим и властвуем).
Делим кучу пополам, потом ещё раз пополам, пока не останутся кучки всего из одного элемента.(не забываем что они уже отсортированы сами по себе). А потом начинаем слияние эти маленькие кучки обратно в большую, но уже правильно раскладывая предметы порядку.

```Python
def merge_sort(arr):  
    if len(arr) > 1:  
        mid = len(arr) // 2  
        left_half = arr[:mid]  
        right_half = arr[mid:]  
  
        merge_sort(left_half)  
        merge_sort(right_half)  
  
        i = j = k = 0  
  
        while i < len(left_half) and j < len(right_half):  
            if left_half[i] < right_half[j]:  
                arr[k] = left_half[i]  
                i += 1  
            else:  
                arr[k] = right_half[j]  
                j += 1  
            k += 1  
  
        while i < len(left_half):  
            arr[k] = left_half[i]  
            i, k = i + 1, k + 1  
  
        while j < len(right_half):  
            arr[k] = right_half[j]  
            j, k = j + 1, k + 1  
  
# Пример использования  
arr = [64, 34, 25, 12, 22, 11, 90]  
merge_sort(arr)  
print("Sorted array:", arr)  

```

*Очень хороший и надёжный способ, но требует места хранения и значительного времени.(Можно сказать улучшенная Вставка).*

2. Подсчётом(Необычный способ).
Этот способ отличается от других тем, что работает тока с числами, если их значения не сильно разнятся.(например оценки учеников,2,3,4,5).Ты просто считаешь, сколько у тебя двоек, сколько троек и т.д. А потом выписываешь их подряд.

```Python
def count_sort(arr):  
    if len(arr) == 0:  
        return  
      
    max_val = max(arr)  
    count = [0] * (max_val + 1)  
      
    # Подсчет количества каждого элемента  
    for num in arr:  
        count[num] += 1  
      
    # Заполнение отсортированного массива  
    sorted_arr = []  
    for i in range(len(count)):  
        while count[i] > 0:  
            sorted_arr.append(i)  
            count[i] -= 1  
      
    return sorted_arr  
  
# Пример использования  
arr = [4, 2, 2, 8, 3, 3, 1]  
print(count_sort(arr))  # Output: [1, 2, 2, 3, 3, 4, 8]  

```

*Срабатывает мгновенно, даже если предметов миллион.*
