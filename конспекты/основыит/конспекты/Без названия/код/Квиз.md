Given a signed 32-bit integer `x`, return `x` _with its digits reversed_. If reversing `x` causes the value to go outside the signed 32-bit integer range `[-231, 231 - 1]`, then return `0`.

```Python
class Solution:
    def reverse(self, x: int) -> int:
        tt = 0
        jj = 2147483648
        uu = -2147483648
        if x < 0:
            tt = int(str(abs(x))[::-1])* -1
        else:
            tt = int(str(abs(x))[::-1])
        if tt  >= jj or tt < uu:
            return 0 
        else:
             return tt
```

Назначаем переменные 
если меньше нуля то проводим его через 1 условие если больше то через 2. Потом сравниваем  и выводим ответ.
