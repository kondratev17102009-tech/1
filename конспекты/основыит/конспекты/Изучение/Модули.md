**Модули** в Python представляют собой файлы с расширением .py, содержащие код, который можно повторно использовать в других программах. Это позволяет структурировать код, улучшать его читаемость и избегать дублирования.

**Подключение модулей**

Для подключения модуля используется оператор _import_. Например:
```Python 
import math
print(math.sqrt(16)) # Вывод: 4.0
```
Можно импортировать конкретные функции или переменные с помощью _from ... import_:
```Python
from math import pi, sin
print(pi) # Вывод: 3.141592653589793
print(sin(0)) # Вывод: 0.0
```
Для сокращения длинных имен модулей можно использовать псевдонимы:
```Python
import math as m
print(m.sqrt(25)) # Вывод: 5.0
```
