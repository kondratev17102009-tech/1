**Рекурсия** - это способ организации кода, при котором функция **вызывает саму себя** для решения подзадачи, похожей на исходную. Ключевые элементы любой рекурсивной функции:

1. **Базовый случай** - условие выхода, при котором функция перестаёт вызывать себя.
2. **Рекурсивный вызов** - обращение к самой себе с изменёнными параметрами, приближающими к базовому случаю.

Без базового случая возникает **RecursionError**, так как Python ограничивает глубину стека вызовов (по умолчанию ~1000).

**Пример: сумма чисел от n до 1**

```Python
def sum_to_one(n):
   if n == 1: # базовый случай
       return 1
   return n + sum_to_one(n - 1) # рекурсивный вызов
print(sum_to_one(5)) # 15
```
