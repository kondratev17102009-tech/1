**Итераторы в Python** - это объекты, которые позволяют перебирать элементы итерируемых объектов, таких как списки, кортежи, строки и множества. Они реализуют протокол итератора, который состоит из двух методов: __iter__()_ и __next__()_.

Итерируемый объект — это объект, который можно передать функции _iter()_, чтобы получить итератор. Итератор, в свою очередь, используется для последовательного извлечения элементов с помощью функции _next()_. Когда элементы заканчиваются, вызывается исключение _StopIteration_.

Пример работы с итератором:

```Python 
numbers = [1, 2, 3]
iterator = iter(numbers)
print(next(iterator)) # 1
print(next(iterator)) # 2
print(next(iterator)) # 3
# Следующий вызов вызовет StopIteration
```
