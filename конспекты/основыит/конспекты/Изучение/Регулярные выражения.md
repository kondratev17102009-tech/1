**Регулярные выражения** (RegEx) - это мощный инструмент для работы с текстом, позволяющий находить, извлекать, заменять и проверять строки на соответствие определённым шаблонам. В Python для работы с регулярными выражениями используется модуль re.

Основные методы модуля re:

**re.match(pattern, string)**: Ищет совпадение только в начале строки. Например:
```Python 
import re
result = re.match(r'AV', 'AV Analytics')
print(result.group(0)) # Вывод: AV
```
**re.search(pattern, string)**: Ищет первое совпадение в любой части строки.
```Python
result = re.search(r'Analytics', 'AV Analytics')
print(result.group(0)) # Вывод: Analytics
```
 **re.findall(pattern, string)**: Возвращает список всех совпадений.
 ```Python 
 result = re.findall(r'AV', 'AV Analytics AV')
print(result) # Вывод: ['AV', 'AV']
 ```
 **re.split(pattern, string, maxsplit=0)**: Разделяет строку по заданному шаблону.
 ```Python
 result = re.split(r'\s', 'AV Analytics Vidhya')
print(result) # Вывод: ['AV', 'Analytics', 'Vidhya']
 ```
 **re.sub(pattern, repl, string)**: Заменяет все совпадения шаблона на указанную строку.
 ```Python
 result = re.sub(r'India', 'World', 'Hello India')
print(result) # Вывод: Hello World
 ```
 **re.compile(pattern)**: Компилирует регулярное выражение для многократного использования.
 ```Python 
 pattern = re.compile(r'AV')
result = pattern.findall('AV Analytics AV')
print(result) # Вывод: ['AV', 'AV']
 ```
