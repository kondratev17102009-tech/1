### Введение. 
**Структура данных** - это программная единица позволяющая хранить и обрабатывать однотипные или логические связанные данные. Для добавления поиска изменения и удаления данных структура данных предоставляет  некоторый набор функции  составляющих её интерфейс. 
### Связный список.
Это базовая динамическая ЗД состоящая из узлов каждая из которых содержит значение и ссылку на следующий узел. 
**Двойной связный список** - это список в котором содержит ссылки  ещё и на предыдущие элемент. 

### Что такое деревья?
**Деревья** - это структура данных расширяющая идею сворованного списка, однако в отличии от них они могут иметь более одного узла. Узлы дерева могут иметь любое количество дочерних узлов. Благодаря этому данные представлены в виде гибкого ветвления.

### Графы
При помощи графов можно расширить область применения "деревьев" и ослабить строгую родительскую связь в них. Узлы не имеют явной иерархии. Каждый из узлом может быть соединён с любым другим узлом.  

class cat_Node:

    #через ключевое слово self объявляются ПОЛЯ класса,
    #т.е. переменные, которые будут принадлежать его экземпляру

    def __init__(self, n = None):
        self.prev = None
        self.next = None
        self.name = n

    def __str__(self):
        return self.name

class cat_LinkedList:

    def __init__(self):
        self.head = None
        self.tail = None

    def __str__(self):

        cats = ""
        current = self.head
        if current is None:
            print("Список пуст.")
            return
        while current is not None:
            cats += current.name + " "
            current = current.next
        return cats

    def add(self, cat_name):

        #Мы создаём переменную, в которой будем хранить ссылку на новый
        #созданный узел. При его создании, мы передаём информацию - имя кота
        newCat = cat_Node(cat_name)

        #Если указатель head для списка - пустой, то мы указываем 
        #нашу ссылку newCat в качестве указателя (строки 25, 26)
        if self.head is None:
            self.head = newCat
        
        #Иначе, если указатель уже есть, мы объявляем переменную current и храним
        #в нём ссылку на "голову" нашего списка. Затем перебираем все элементы 
        #списка до тех пор, пока указатель на следующий элемент не станет пустым.
        #И добавляем в качестве пустого ссылку на новый элемент.
        else:
            current = self.head
            while current.next is not None:
                current = current.next
            current.next = newCat
