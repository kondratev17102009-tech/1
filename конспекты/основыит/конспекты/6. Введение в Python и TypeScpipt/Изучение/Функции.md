**Функции в Python** - это именованные блоки кода, которые позволяют объединять инструкции, пере использовать их и делать программу более читаемой. Они могут быть как встроенными (например, print, len, sum), так и пользовательскими, созданными с помощью ключевого слова def.

```Python
def multiply_by_two(a):
   result = a * 2
   return result
print(multiply_by_two(5)) # Выведет 10
```

Здесь multiply_by_two - имя функции, a - параметр, return возвращает результат.

Типы аргументов:

**Позиционные** - передаются в порядке объявления параметров.
**Именованные** - указываются с присвоением (_b=3_), порядок не важен. Можно
**комбинировать**: сначала позиционные, затем именованные.
