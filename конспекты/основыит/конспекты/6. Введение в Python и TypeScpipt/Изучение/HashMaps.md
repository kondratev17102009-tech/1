HashMap, HashTable, Map, Dictionary или Ассоциативен - все имена одной и той же структуры данных. Это структура данных, реализующая абстрактный тип данных множества - структуру, способную сопоставлять ключи с значениями.

Основы работы:
Хэш-функция преобразует ключ в уникальное числовое значение, которое используется как индекс для ранения значения в хэш- таблице. Это позволяет быстро находить, добавлять и удалять данные по ключу. 

Свойства:

- Уникальность ключе. Ключи должны быть неизменяемыми.
- Уникальность значения. Если ключ уже существует, его значение перезаписывается. 
- Демонический размер. Хэш-карты автоматически адаптируются к изменению объёма данных.
- Сохранение порядка элементов. Наличия с Python 3.7, словари сохраняют порядок добавления ключей.
