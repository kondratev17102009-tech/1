**Лямбда-функции** - это анонимные функции в Python, которые создаются с помощью ключевого слова lambda и не требуют явного объявления через def_. Они часто используются для кратких операций, особенно в сочетании с функциями высшего порядка.

**Использование с функциями высшего порядка** Лямбда-функции часто применяются с map(), filter() и reduce().

```Python
# filter: оставить только четные числа
nums = [1, 2, 3, 4, 5]
even_nums = list(filter(lambda x: x % 2 == 0, nums))
print(even_nums) # [2, 4]
# map: возвести в квадрат
squares = list(map(lambda x: x ** 2, nums))
print(squares) # [1, 4, 9, 16, 25]
# reduce: сумма элементов
from functools import reduce
total = reduce(lambda a, b: a + b, nums)
print(total) # 15
```
