**Декораторы** - это мощный инструмент Python, который позволяет изменять или расширять поведение функций или классов без изменения их исходного кода. Они принимают функцию или класс, добавляют к ним дополнительную функциональность и возвращают модифицированный объект.

Декоратор — это функция, которая принимает другую функцию в качестве аргумента, оборачивает её в дополнительную логику и возвращает новую функцию. Пример простого декоратора:

```Python 
def decorator(func):
   def wrapper():
       print("Перед вызовом функции")
       func()
       print("После вызова функции")
   return wrapper
@decorator
def say_hello():
   print("Привет, мир!")
say_hello()
```
