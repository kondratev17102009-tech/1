```Python
is_running = True

  

while is_running:

    raw = input(" jrfr")

    if raw == "exit":

        is_running = False

        print("fdsfs")

    elif raw == "Продолжить":        

     x = int(input())

     y = int(input())

     z = 0

    try:

        z = x/y

    except ZeroDivisionError:

            print("Исключение")

    else:

        print("Некоректный код")
```

Этот код представляет собой цикл, который продолжает выполняться, пока переменная `is_running` равна `True`.

Исключения в Python - это события возникающие при возникновении какой либо нарушения потока выполнения программного кода и его работа. В 1 очередь они позволяют позволять работу после обнаружения проблемы или неисправности в коде или работе кода(а не аварийное завершить его).

Некоторые Встроенные исключения:

- AssertionError - возникновение при ошибке когда assert не повезло.
- AttributeError - попытка доступа к атрибуту.
- EOFEror - возникает когда функция input достигает конца файла.
