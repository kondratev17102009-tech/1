В Python переменные создаются автоматически при присвоении значения, и не требуется явного объявления типа. Для удобства чтения и поддержки кода соблюдаются правила именования: использовать буквы, цифры и символ _. Имя не должно начинаться с цифры. Python использует динамическую типизацию, поэтому переменная может менять тип в процессе выполнения. Переменные могут быть локальными или глобальными: локальная переменная создается внутри функции и доступна только внутри неё, а глобальная объявляется вне функций и доступна везде. Для изменения глобальной переменной внутри функции используется ключевое слово global: global x x += 1 print (x) # 6.
