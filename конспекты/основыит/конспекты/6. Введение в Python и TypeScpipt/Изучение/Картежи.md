**Кортежи** (tuple) - это неизменяемые последовательности, которые позволяют хранить упорядоченные коллекции элементов. Они схожи со списками, но их содержимое нельзя изменять после создания, что делает их полезными для хранения данных, которые должны оставаться неизменными.

```Python
# Пустой кортеж
empty_tuple = tuple()
print(empty_tuple) # Вывод: ()
# Кортеж из нескольких элементов
example_tuple = (1, 2, 3)
print(example_tuple) # Вывод: (1, 2, 3)
# Кортеж из строки
string_tuple = tuple("hello")
print(string_tuple) # Вывод: ('h', 'e', 'l', 'l', 'o')
# Кортеж из списка
list_tuple = tuple([1, 2, 3])
print(list_tuple) # Вывод: (1, 2, 3)
```
Преимущества кортежей:

1. **Неизменяемость**: Кортежи защищены от изменений, что делает их безопасными для хранения неизменяемых данных.
2. **Производительность**: Они занимают меньше памяти и быстрее обрабатываются по сравнению со списками.
3. **Использование в качестве ключей словаря**: Кортежи могут быть ключами в словарях, так как они неизменяемы.
