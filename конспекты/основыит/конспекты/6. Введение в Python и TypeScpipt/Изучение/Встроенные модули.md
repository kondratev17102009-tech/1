**В Python модуль** - это отдельный файл с расширением _.py_, содержащий функции, классы, переменные и исполняемый код, который можно **импортировать** в другие программы. Модули помогают структурировать проект, повторно использовать код и избегать конфликтов имён.

**Пример создания модуля** Создадим файл mymath.py:
```Python
# mymath.py
"""Модуль с математическими функциями."""
PI = 3.14159
def add(a, b):
   return a + b
def divide(a, b):
   if b == 0:
       raise ValueError("Деление на ноль невозможно")
   return a / b
```
Некоторые основные модули в Python:

- math - доступ к математическим функциям.
- random - функция для генерации случайных чисел или ещё чего-то.
- os - инструмент для взаимодействия с ОС.
- sys - обеспечат доступ к параметрам и функциям ОС.
- re  - модуль для работы с регулярными.
- datetime - модуль для работы с временем.
- json - используется для работы с JSON, сериализации и десериализации. 
- collections - спец данные: счётчики, очереди, словари.
- hashlib - интерфейс для безопасности алгоритмов.
