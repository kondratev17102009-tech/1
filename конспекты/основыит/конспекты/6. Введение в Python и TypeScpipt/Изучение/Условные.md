**Условные операторы** позволяют выполнять различные действия в зависимости от выполнения определённых условий. Они являются основой для управления потоком выполнения программы и широко используются в большинстве языков программирования.

**Основные конструкции условных операторов**:
В большинстве языков программирования условные операторы представлены конструкциями _if_, _else if_ (или _elif_), и _else_. Они позволяют проверять одно или несколько условий и выполнять соответствующий блок кода.

Пример на Python:
```Python 
x = int(input("Введите число: "))
if x > 0:
   print("Число положительное.")
elif x < 0:
   print("Число отрицательное.")
else:
   print("Это ноль.")
```

**Объяснение:**

- if проверяет первое условие.
- elif используется для проверки дополнительных условий.
- else выполняется, если ни одно из условий не оказалось истинным.

**Логические операторы**

Для объединения нескольких условий используются логические операторы:

- and - логическое И (оба условия должны быть истинными).
- or - логическое ИЛИ (хотя бы одно условие должно быть истинным).
- not -логическое НЕ (инвертирует значение условия).
