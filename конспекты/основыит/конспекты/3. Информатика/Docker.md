#Деплой #Docker #Kubernetes #Основы  
**Деплой** - процесс развёртывания приложения на холстине для обеспечения внешнего доступа. 

Образ - 
Контейнер - 
**Docker** - инструмент для контейнеризации, который упаковывает приложение с зависимостями в лёгкий изолируемый контейнер. Изолирует процессы на уровне ядра. Работает на одном холсте. Решает проблемы совместимости упрощает деплой и воспроизводимость. 

**Kubernetes** - инструмент для управления контейнерами обеспечивающий высокую эффективность и отказоустойчивость развёрнутых систем. 

Основные команды:
```Dockerfile
#База докера - образ

FROM python:3.11-slim

#Рабочая директория

WORKDIR / app

#Установка зависимости

RUN pip install poetry 

COPY pyproject.toml poetry.lock ./
```
