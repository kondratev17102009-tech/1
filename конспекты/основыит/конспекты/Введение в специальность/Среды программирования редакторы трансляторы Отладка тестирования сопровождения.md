### Среда программирования 

СП - это экосистем интегрирующая аппаратные и программные компоненты
Для разработки ПО. Согласно определению в стандарте  ISO/IEC  2382 - 1  это совокупность средств обеспечивающих создание модификацию и создание программ. 

### Что входит в среду 

IDE 
Платформа объединяющая редактор компилятор и отладчик 
Терминалы представляют - (Командная строка ) из себя программы для выполнения тактовых команд 

Библиотеки и фреимворки 

Библиотека представляет собой набор готовых функций для решения задач 
Фреимворк - это каркас или шаблон предоставляет готовый набор инструментов библиотек и правил для ускорений разработки прел.

Облачные сферы - онлайн платформа для совместного работы 

Системы автоматической сборки -  онлайн платформа для современной работы система автоматической сборки 

### Редакторы 

программные инструменты для манипуляции исходного кода 
Формально они реализуют поддержку
