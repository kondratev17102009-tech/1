```Python 
class Solution:
    def pivotInteger(self, n: int) -> int:
        res0 = (n* (n+1) //2)
        res1 = int(res0 ** 0.5)

        if res1 * res1 == res0:
            return res1

        return -1    
```
В **Power Pivot** вы можете использовать **DAX-формулы** для анализа данных. Вот основные моменты:

**Вычисляемые столбцы** и **меры** используются для выполнения вычислений и фильтрации данных.

Формулы DAX похожи на формулы Excel, но DAX позволяет выполнять более сложные вычисления и работы с реляционными данными.]

Вы можете создавать меры с помощью агрегатных функций, таких как SUM или AVERAGE, которые автоматически создадут формулу.
Для создания вычисляемого столбца или меры введите формулу в соответствующий столбец, например: `SUM('Таблица'[Продажи])` для суммирования продаж.

Для более подробной информации о DAX-формулах и их использовании в Power Pivot, вы можете обратиться к Microsoft Support и Techy.
