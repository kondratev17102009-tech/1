RDP -  это (Remote Desktop Protocol ) протокол для удалённого подключения к другим устройствам. 

1. Действие убедится что у вас есть права админа.
2. Параметры - Система - о системе(запоминаем данные).
3. Ищем в Windows - подключение к удалённому рабочему столу.
4. Вводим IP адрес нужного нам устройства. 
5. Люто взламываем устройство.

**Virtual Network Computing** (**VNC**)  - система удалённого доступа использующая RFB (remote framebuffer).
Система VNC платформ независима: VNC-клиент, называемый VNC viewer, запущенный на одной операционной системе может подключаться к VNC-серверу, работающему на любой другой ОС, К одному VNC.
