### 1. Что такое BAT-файл?

**BAT-файл** — это текстовый файл с расширением `.bat`, который содержит набор команд для командной строки Windows . При его запуске эти команды выполняются автоматически и последовательно.

**Для чего используется:**

- **Автоматизация рутины:** копирование, перемещение, резервное копирование файлов
    
- **Массовый запуск:** открытие нескольких программ или сайтов одним кликом
    
- **Настройка системы:** изменение сетевых параметров, очистка временных файлов

### 2. Создание и синтаксис BAT-файла

####  Как создать:

1. Откройте **«Блокнот»** (Notepad).
    
2. Напишите команды, каждую с новой строки.
    
3. Нажмите **«Файл»** → **«Сохранить как»**.
    
4. Выберите тип файла **«Все файлы»**, укажите имя, обязательно добавив расширение `.bat` (например, `myscript.bat`) 
#### Основные команды для BAT-файлов:

|Команда|Что делает|Пример использования|
|---|---|---|
|`@echo off`|Скрывает служебные сообщения, оставляя только результат.|`@echo off`|
|`echo`|Выводит текст на экран.|`echo Привет, мир!`|
|`pause`|Останавливает выполнение и ждет нажатия клавиши (чтобы окно не закрылось сразу).|`pause`|
|`start`|Запускает программу или открывает папку/сайт.|`start notepad.exe`|
|`cd`|Меняет текущую папку.|`cd C:\Users\Public`|
|`md` или `mkdir`|Создает новую папку.|`md NewFolder`|
|`copy`|Копирует файлы.|`copy file.txt D:\backup\`|
|`del`|Удаляет файлы.|`del C:\Temp\*.tmp`|
|`color`|Меняет цвет фона и текста консоли.|`color 0A` (черный фон, зеленый текст)|
|`rem` или `::`|Добавляет комментарий (не выполняется).|`rem Это комментарий`|

#### Переменные:

Используются для хранения данных. Объявляются через `set`, а вызываются через знаки процента `%` 

batch

set name=Алексей
echo Привет, %name%!

### 3. Утилита WinGet (Диспетчер пакетов Windows)

**WinGet** — это официальная утилита от Microsoft для командной строки, которая позволяет искать, устанавливать, обновлять и удалять программы автоматически.

####  Где используется:

Входит в состав современных Windows 10 (1809+) и Windows 11. Если команда `winget` не работает, её можно установить из Microsoft Store (компонент «Установщик приложений») 
####  Основные команды WinGet:

|Команда|Действие|Пример|
|---|---|---|
|`winget search`|Поиск программы в репозитории.|`winget search telegram`|
|`winget install`|Установка программы.|`winget install Google.Chrome`|
|`winget list`|Показать список установленных программ.|`winget list`|
|`winget upgrade`|Обновить программу (или все, если добавить `--all`).|`winget upgrade --all`|
|`winget uninstall`|Удалить программу.|`winget uninstall "7zip"`|
|`winget show`|Показать подробную информацию о пакете.|`winget show Microsoft.PowerShell`|
|`winget source`|Управление источниками (репозиториями) пакетов.|`winget source list`|

#### Важные особенности:

- **ID пакета:** Вместо названия программы лучше использовать её уникальный ID (например, `Microsoft.PowerShell`), чтобы избежать ошибок.
    
- **Права администратора:** Для установки большинства программ терминал нужно запускать **от имени администратора**
