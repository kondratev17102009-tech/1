```python

#x,y,z,w = 0,0,0,0

  

for x in [0,1]:

    for y in [0,1]:

        for z in [0,1]:

            for w in [0,1]:

             if ((x or y) and not(y == z) and not (w)) == 1:

              print (x,y,z,w)
```


Решение:
Задачу решается в 2 этапа:
написать код на языке программирования в моём лучи Питон 
и подставить полученные значения в таблицу согласно задачи их порядку.

Данный код позволяет указанную задачу и схожие по типу. Работает по принципу перебора всех возможных значений для указанных переменных в логическом выражении. Так, сначала $x =0 , y=0 ....$  Затем значение z при интеграции цикла увеличивается на 1. Затем для остальных значений происходит тоже самое. При этом когда значение верхнего цикла(родительского) увеличивается на один весть код внутри повторяется 

Когда лог. выражение при подстановке знач. переменных будет возвращать нужно логическое значение (истина в нашем случаи) мы выведем значение этих переменных на экран. 

Полученный результат работы значения можно логически расставить по своим местам в таблице, опираясь на кол. 0 и 1 в столбце также опирались на значение переменных в строках.
