 
``` python
import random 

ANSWER = ['да', 'нет']

while True:
    x = input("Введите текст")

    if x.find("выход") !=  -1:
    break
     else:
      print(random.choice(ANSWER))
```

Код создает бесконечный **цикл**, который:

1. **Запрашивает** у пользователя ввод текста.
2. **Проверяет**, содержит ли введенный текст слово “выход” (независимо от регистра).
3. Если “выход” **найден**, цикл **прерывается** (`break`).
4. В противном случае, он **случайным образом выбирает** и выводит одно из слов: ‘да’ или ‘нет’.
