Цикл while -  проверяет истинность условия.

Цикл while - Да, совершенно верно, цикл while (с англ. "пока") в программировании проверяет истинность некоторого условия перед каждой итерацией и выполняет блок кода до тех пор, пока это условие остается истинным (возвращает true). Если условие изначально ложно, тело цикла не выполнится ни разу.

Цикл For - это цикл наборы значений который в последствии пробегает по набору  данных.(окак)

Цикл For - это управляющая конструкция в программировании, которая позволяет повторять определенный блок кода для каждого элемента в заданной последовательности или заданное количество раз.(окак)

Цикл continue - выполняет переход к следующей итерации цикла.

Цикл continue - Оператор continue в программировании пропускает оставшуюся часть текущей итерации цикла (while, for, do-while) и передает управление к началу следующей итерации, а не завершает весь цикл, как break. Он полезен для исключения определенных элементов или условий из обработки без остановки всего цикла

Цикл Break - break осуществляет выход из цикла.

Цикл Break - Оператор break в программировании прерывает выполнение ближайшего внутреннего цикла (например, for, while или do-while) и передает управление оператору, следующему за этим циклом. Он позволяет досрочно выйти из цикла, даже если его условие еще истинно, что полезно для оптимизации или когда условие прерывания находится внутри тела цикла. 

Цикл rainge() - генерирует числовую последовательность 

Цикл rainge()  -  Функция range() в Python генерирует последовательность целых чисел и чаще всего используется в for циклах для выполнения кода определенное количество раз или итерации по индексам. Она имеет три формы: range(stop) для последовательности от 0 до stop - 1, range(start, stop) от start до stop - 1, и range(start, stop, step) для создания последовательности с заданным шагом.
