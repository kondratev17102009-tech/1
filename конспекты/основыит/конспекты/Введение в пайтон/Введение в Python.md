Алгоритм 
Понятие алгоритма 
тип алгоритма 

Алгоритм это последовательность шагов или инструкции. необходимых для выполнения задач как и информатика алгоритм может быть представлен в различных формах - текст описание псевдокод блок схемы программный код.

Характер Алгоритма 

Определённость - каждый шаг алгоритма должен быть чётко определён и не вызывать двусмысленность.

Конечность - алгоритм должен завершится после конечного числа шагов 

Входные и выходные 

Эффективность - алгоритм должен быть достаточно эффективен по времени  и ресурсам.

Виды алгоритмов:

Линейный - Каждый шаг выполнятся один за другим(последовательно). Не содержат циклов или условий 

Разветвляющая - Использует условия для принятия решения и в зависимости от них алгоритм может следовать различным путям 

Циклический алгоритм - Включает в себя повторяющиеся действия , которые выполняется до тех пор, пока не будет соблюдено определённое условие.

Новое в Python

Переменная - именования область памяти в компьютере в которой записаны какие либо данные. Значение переменной называется литералом и может представлять несколько типов в данных. 

Чтобы объявить переменную в Python используется оператор присваивания (=).
В левой части имя переменной в правой её литерал.

Блок схемы в программирование.

Это графические представление алгоритма или процесса которое показывает последовательность действий и логику выполнения программы. В контексте программирования помогает визуализировать структуру кода тем самым упрощая разработку алгоритма тестирование.

Основные  элементы блок схемы 

Овал - Эта начало или конец алгоритма 

Прямоугольник -  1 процесс - выполнение операции или действия например  присвоение значения переменной вызов функции и т.д.

Параллелограмм(Ввод/вывод) - операция ввода и вывода данных.

Ромб(условие) - проверка условия, ветвление 

Стрелки - направленные потока выполнения программы.
