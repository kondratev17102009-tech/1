### Введение
Класс - это пользовательский тип данных  конструкция ООП
(Объектно-Ориентированное Программирована ) которая описывает тот или иной объект его свойства и способы взаимодействия с ним.

Класс имеет поля - переменные коротыше будут принадлежать к его экземпляру а также методы - функции которые также можно будет использовать для его экземпляров.

Простыми словами класс это шаблон для реализации какого-то объекта.
Например мы знаем как должна выглядеть кружка какие характеристики она имеет. При помощи класса мы описываем абстрактную маму кружку а экземпляры будут является настоящими конкретными кражами с определёнными  характеристиками и действиями мамы-кружки. 

Класс решает несколько важных задач:
- дублирование кода 
- Унификация объектов (Т.е. одинаковые по задумке объекты действительно будут иметь одинаковые характеристики и функционал)

### Классы в Python

Объявляются при помощи ключевого слова class и указании имени класса. Затем ставится : и за тем со следующей строки описывается тело класса. 
 
 Чтобы объявить поля класса используется ключевое слово self после которого через точку указывается имя поля. Поле = Переменная .(Поле = переменная внутри класса ). Однако через self можно обращаться только из тела метода. В параметрах метода также обязательно должен быть указан метод self.

Для инициализации полей при создании экземпляра класса  используется функция конструктор __init__ она также обязательно принимает параметр self.  Эта функция будет автоматически вызвана при создании экземпляра класса. Если она не объявлена будет использован конструктор по умолчанию (пустой).

Создание экземпляра класса(объект класса ) происходит вне тела класса при помощи указания имени класса и круглых скобок после имени. Если в конструкторе кроме self указаны другие параметры то при создании объекта в скобках мы также укажем атрибуты - то что является параметрами описании функции.
