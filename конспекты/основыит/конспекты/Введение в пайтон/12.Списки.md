## Определение
Списки в Python - это упорядоченные изменяемые коллекции произвольных типов (почти как массив, но в отличие от него, значение в списках могут отличаться). Они создаются с помощью квадратных скобок или функции-конструктора `list()`. К каждому элементу можно обратиться по его индексу.

Списки не обязательно должны содержать только однотипные объекты.

```python
o = []
q = list()
oq = ["pavuk", True, 3]
qo = list(oa)
```

Функция `list()` принимает значение итерируемого типа, т.е. другие списки, строки и т.д.

```python
oreshki_big_bob = ["cheese", "bacon", "onion"]
oreshki_big_bob[2]
oreshki_big_bob[0:2] # Срез
oreshki_big_bob[0, 2, 1] # Начало, конец, шаг 
```

Срезы - это способ получения части списка при помощи указания индексов. Готовый срез является новым списком. Для создания среза указывается имя списка и определённые индексы. Всего есть 3 способа указать правила генерации срезов:

1. Указать конечное значение, т.е. будет сгенерирован срез от нуля до указанного (не включительно).
2. Указать начальное и конечное значения.
3. Указать начальное и конечное значения

Если требуется использовать один из указанных способов, но отсутствует некоторая информация, в квадратных скобках можно поставить двоеточие вместо пропущенного значения.

```python
a = ["literally", "me", "and"]

```

## Методы работы со списками
#### append() 

Сначала указывается имя списка, затем через точку вызывается метод `append()`, в который передаётся элемент `item` для добавления. Элемент добавляется в конец списка.

```python
peanut = [1, 2, 3]
peanut.append("4")

print(peanut)
```

#### index(item)
**Индекс конкретного элемента**
 
Сначала указывается имя списка, через точку вызывается метод `index`, в который подаётся элемент item. В результате работы возвращает индекс указанного предмета. Если элементов с одинаковым значением несколько, возвращает индекс первого найденного в списке. Если элемента нет, выбрасывает исключение ValueError.

```python
peanut = [1, 2, 3]

print(peanut.index(2)) #3
print(peanut.index(5)) #ValueError
```

#### len(list)  
**"Длина списка"**

Сначала вызывается функция `len`, а ей в качестве аргумента передаётся конкретный список. Возвращает длину списка в виде целого числа.

```python
peanut = [1, 2, 3]

print(len(peanut)) #3
```

#### min(list) и max(list)
**Минимальное и максимальное значение в списке**

Функции возвращают минимальное или максимальное значение из указанного в качестве аргумента списка.

```python
peanut = [1, 2, 3]

print(max(peanut)) #3
print(min(peanut)) #1
```

#### sorted(list)
**Возвращает новый отсортированный список**

Функция возвращает новый список, содержащий все элементы исходного списка, но в отсортированном порядке (по возрастанию, если не указано иное). Оригинальный список не изменяется.

```python
peanut = [3, 1, 2]
sorted_peanut = sorted(peanut)

print(peanut)          #[3, 1, 2] (Оригинальный список не изменился)
print(sorted_peanut)   #[1, 2, 3]
```

#### list.insert(index, element)
**Вставляет элемент на указанную позицию**
Метод вставляет element в список перед указанным `index`. Элементы сдвигаются вправо.

```python
peanut = ['a', 'c']
peanut.insert(1, 'b') 

print(peanut) #['a', 'b', 'c']
```

#### list.extend(iterable)
**Добавляет элементы итерируемого объекта**

Метод расширяет список, добавляя в его конец все элементы из указанного итерируемого объекта (например, другого списка). Похож на оператор `+` или `+=`, но выполняется на месте.

```python
peanut = [1, 2]
other_list = [3, 4]
peanut.extend(other_list)

print(peanut) #[1, 2, 3, 4]
```

#### list.remove(element)
**Удаляет первое вхождение элемента**

Метод удаляет первое вхождение указанного `element` из списка. Если элемент не найден, возникает ошибка `ValueError`.

```python
peanut = [10, 20, 10]
peanut.remove(10)

print(peanut) #[20, 10]
```


#### list.pop([index])
**Удаляет и возвращает элемент**

Метод удаляет элемент по указанному index и возвращает его значение. Если `index` не указан, удаляет и возвращает последний элемент списка.

```python
peanut = ['a', 'b', 'c']
last_element = peanut.pop()   # Удаляет 'c'

print(last_element)           #c
print(peanut)                 #['a', 'b']
```


#### list.count(element)
**Считает количество вхождений**

Метод возвращает количество раз, которое указанный element встречается в списке.

```python
peanut = [1, 2, 1, 3]
count_of_one = peanut.count(1)

print(count_of_one) #2
```

#### list.sort(reverse=False)
**Сортирует список на месте**

Метод сортирует сам список (по возрастанию по умолчанию) и не возвращает никакого значения (`None`). Используйте reverse=True для сортировки по убыванию.

```python
peanut = [3, 1, 2]
peanut.sort() 

print(peanut) #[1, 2, 3]
```

#### list.reverse()
**Разворачивает порядок элементов**

Метод изменяет порядок элементов в списке на обратный на месте.

```python
peanut = [1, 2, 3]
peanut.reverse()

print(peanut) #[3, 2, 1]
```

#### list.copy()
**Создает поверхностную копию списка**

Метод возвращает новую поверхностную копию списка. Это важно для того, чтобы изменения в новом списке не затрагивали оригинал.

```python
original = [1, 2]
copy_of_list = original.copy()
copy_of_list.append(3)

print(original)     #[1, 2]
print(copy_of_list) #[1, 2, 3]
```
